Swift Translator Automated Testing (Playwright)
Student ID : IT23207554

This project contains automated test cases for the Swift Translator Web Application using Playwright.
The tests validate positive and negative functional scenarios and basic UI behavior for Sinhala-English translation.
Its contain 24 positive test cases and 10 negative test cases and 1 UI test 

Technologies Used:
Node.js
Playwright Test Framework
Javascript
Git & Github

Project Structure:
project-root/
│
├── tests/
│   └── it23207554.spec.js
├── package.json

└── README.md

Prerequisites:
Node.js (v16 or above)
Git 
npm
Playwright

Installation
------------
1) Clone the repository:
   git clone https://github.com/HashinthaHashan/IT23207554.git

2) Navigate to the project folder:
   cd IT23207554

4) Install dependencies:
   npm install

5) Install Playwright browsers:
   npx playwright install

Run the Tests
-------------
npx playwright test tests/it23207554.spec.js
